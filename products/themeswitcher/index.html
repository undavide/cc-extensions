Theme Switcher

Change Photoshop GUI Theme

BY DAVIDE BARRANCA

Photoshop GUI

Since CS6, Photoshop supports four Graphic User Interface styling – that is: the interface color – from light to dark: Original, Light Gray, Medium Gray, Dark Gray. Theme Switcher is a handy extension with a minimal interface that lets you cycle through them with one click only (no need to dig in the Preferences anymore).
By Davide Barranca.
